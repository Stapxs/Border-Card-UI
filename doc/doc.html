卡片：ss-card

Border Card UI 顾名思义组成依赖大量的卡片，卡片组件是最基础的页面组件。

基本使用
HTML
                                <div class="ss-card"></div>

你可以把卡片相互叠起来，但是 BCUI 并不会自动处理卡片之间的层级颜色，所以你需要自己处理。

BCUI 提供最大两层的卡片颜色以及对应的字体颜色，同时还提供了用于叠加在主题色上的反转字体颜色。

  --color-bg
  --color-card
  --color-card-1
  --color-card-2
添加标题

你可以使用<header>来为卡片添加标题。

HTML
                                <div class="ss-card">
    <header><div></div>带有标头的卡片</header>
</div>
<div class="ss-card">
    <header>
        <i class="fa fa-file-code-o" aria-hidden="true"></i>
        带有文本图标的卡片
    </header>
</div>
语录栏：ana-card

这是个卡片扩展，用于展示一个单行的重要文本，可以用于公告等。

基本使用
HTML
                                <div class="ss-card ana-card">
    <div></div>
    <span>『我的世界』 —— 很简单而已</span>
</div>

单选按钮：ss-radio

就只是普通的单选按钮而已。

基本使用

你需要覆盖--radio-size参数来修改它的大小，默认为32px，见下面的例子：

HTML
                                <label class="ss-radio" style="--radio-size: 23px;">
    <input type="radio">
    <div><div></div></div>
</label>

多选按钮：ss-checkbox

就只是普通的多选按钮而已，为了符合配色要求进行简单重制的版本。

基本使用

你可以使用--checkbox-size参数来修改它的大小，默认为13px，见下面的例子：

HTML
                                <label class="ss-radio" style="--checkbox-size: 15px;">
    <input type="checkbox">
    <div></div>
</label>
VIEW
